Если двойной клик по BAT/CMD ничего не показывает

1. Полностью распакуйте архив.
   Не запускайте файлы из окна WinRAR.

2. Откройте папку "Мои решения".

3. Кликните правой кнопкой мыши по пустому месту в папке.
   Выберите "Открыть в Терминале" или "Open in Terminal".

4. Введите команду:

   RUN_DIAGNOSTIC.cmd

5. Если диагностика открылась, затем запустите:

   RUN_PROJECT_SAFE.cmd

6. Если Windows пишет, что запуск заблокирован:
   - правый клик по файлу RUN_PROJECT_SAFE.cmd;
   - Свойства;
   - Разблокировать;
   - ОК.

7. Если Python не найден, установите Python 3.11+ с python.org.
   Во время установки обязательно поставьте галочку:
   Add Python to PATH.
