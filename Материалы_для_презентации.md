# Материалы для презентации

## Слайд 1. Титульный

Тема: Искусственная иммунная сеть для задачи поиска кратчайшего пути в графе.  
Вариант 3.  
Сидоренко Максим.

## Слайд 2. Постановка задачи

Есть взвешенный неориентированный граф G = (V, E). Нужно найти путь минимальной суммарной длины от начальной вершины s до конечной вершины t.

Критерии оценки:
- время выполнения;
- длина найденного пути;
- отклонение от эталонного решения Дейкстры;
- динамика целевой функции по итерациям.

## Слайд 3. Идея искусственной иммунной сети

Биологическая аналогия:
- антиген - задача;
- антитело - кандидат-решение;
- аффинность - степень качества решения;
- клон - копия хорошего решения;
- мутация - изменение решения;
- память - сохранение лучших решений.

## Слайд 4. Адаптация к кратчайшему пути

В этой работе антитело кодируется как список вершин, образующих путь от start до target. Целевая функция - сумма весов рёбер пути. Чем меньше сумма, тем лучше решение.

## Слайд 5. Схема алгоритма

1. Создать начальную популяцию путей.
2. Оценить длину каждого пути.
3. Выбрать лучшие пути.
4. Создать клоны лучших путей.
5. Выполнить гипермутацию клонов.
6. Исправить некорректные пути.
7. Выполнить супрессию похожих решений.
8. Обновить иммунную память.
9. Повторить до достижения лимита итераций.

## Слайд 6. Программная реализация

Основные модули:
- graph_model.py - граф, генерация, загрузка;
- dijkstra_baseline.py - эталон Дейкстры;
- immune_network.py - ИС;
- visualization.py - графики и визуализация;
- experiments.py - эксперименты.
