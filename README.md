**Дисциплина:** Генетические алгоритмы и иммунные системы  
**Вариант:** 3 - поиск кратчайшего пути в графе  
**Исполнитель:** Сидоренко Максим  
**Роль в парной работе:** реализация алгоритма искусственной иммунной сети. Генетический алгоритм реализует второй участник пары.

## 2. Установка зависимостей

Из папки `Генетические алгоритмы/Мои решения/` выполнить:

```bash
pip install -r requirements.txt
```

## 3. Быстрый запуск демонстрации

```bash
python main.py
```

После запуска будут обновлены:

```text
results/path_visualization.png
results/objective_vs_iterations_demo.png
results/demo_summary.json
```

В консоли будет выведено:

```text
- количество вершин и рёбер;
- стартовая и конечная вершина;
- путь, найденный искусственной иммунной сетью;
- длина найденного пути;
- эталонный путь по алгоритму Дейкстры;
- отклонение от эталона;
- время выполнения.
```

## 4. Запуск экспериментальной серии

## 6. Формат входного графа

```json
{
  "vertices": 8,
  "start": 0,
  "target": 7,
  "edges": [
    [0, 1, 4],
    [0, 2, 3],
    [1, 3, 2],
    [2, 3, 5],
    [3, 7, 4]
  ]
}
```

Каждое ребро задаётся тройкой `[u, v, weight]`, где `u` и `v` - номера вершин, `weight` - положительный вес ребра.

## 7. Как объяснять алгоритм на защите

- **Антиген** - задача поиска пути в конкретном графе от стартовой вершины к конечной.
- **Антитело** - один возможный путь между стартом и финишем.
- **Аффинность** - качество антитела. Чем короче путь, тем выше качество.
- **Клонирование** - создание копий лучших путей.
- **Гипермутация** - перестройка части пути: замена хвоста, добавление обходного участка или сокращение участка.
- **Супрессия** - удаление слишком похожих решений, чтобы сохранить разнообразие популяции.
- **Иммунная память** - сохранение лучших найденных путей.
- **Дейкстра** - эталонный алгоритм для проверки качества, а не основной сдаваемый метод.

## Красивый веб-интерфейс

Добавлен полноценный Web GUI:

```text
RUN_WEB_GUI.cmd
```

Он запускает локальный сайт:

```text
http://127.0.0.1:5000
```

В интерфейсе можно:

- загрузить демонстрационный граф;
- сгенерировать новый граф;
- выбрать число вершин, рёбер, итераций, размер популяции и вероятность мутации;
- запустить искусственную иммунную сеть;
- увидеть найденный путь прямо на графе;
- сравнить длину пути с алгоритмом Дейкстры;
- увидеть график целевой функции по итерациям;
- запустить серию экспериментов;
- посмотреть таблицу результатов и график времени.

Это основной способ демонстрации проекта на защите.
